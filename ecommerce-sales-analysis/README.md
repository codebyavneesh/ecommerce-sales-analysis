🛒 Ecommerce Sales Analysis
A data analysis project that explores and visualizes sales data from a superstore dataset using Python.

📁 Project Structure
ecommerce-sales-analysis/
│
├── main.py                        # Main Python script
├── superstore.csv                 # Dataset
├── screenshots/                   # Output charts folder
│   ├── Regional_Sales.png
│   ├── category_wise_sales.png
│   ├── monthly_trend.png
│   ├── sub_category_wise_sales.png
│   └── yearly_trend.png
├── .gitignore
└── README.md

📊 Analysis Performed

Regional Sales — Sales distribution across different regions
Category Wise Sales — Sales breakdown by product category
Sub-Category Wise Sales — Detailed sub-category level analysis
Monthly Trend — Month-wise sales trend over time
Yearly Trend — Year-wise sales performance

🛠️ Technologies Used

Python 3.13
Pandas — Data manipulation and analysis
NumPy — Numerical computations
Matplotlib — Data visualization
Seaborn — Statistical data visualization


⚙️ Setup & Installation
1. Clone the repository
bashgit clone https://github.com/codebyavneesh/ecommerce-sales-analysis.git
cd ecommerce-sales-analysis
2. Create virtual environment
bashpython -m venv venv
3. Activate virtual environment
bash# Windows
venv\Scripts\activate
4. Install dependencies
bashpip install numpy pandas matplotlib seaborn
5. Run the project
bashpython main.py

📂 Dataset
The project uses the Superstore Sales Dataset (superstore.csv) which contains sales records including:

Sales, profit, and discount data
